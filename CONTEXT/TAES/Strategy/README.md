## Strategy 内容规范

### 什么属于 Strategy？

| 类型 | 示例 | 变化周期 |
|------|------|----------|
| **核心洞察** | 行业痛点识别、技术趋势判断 | 2-5 年 |
| **市场定位** | "我们是什么"、差异化定位 | 1-3 年 |
| **ICP** | 理想客户画像、目标细分市场 | 1-2 年 |
| **竞争壁垒** | 护城河、不可替代性 | 2-5 年 |

### 什么不属于 Strategy？

| 类型 | 应归属 | 原因 |
|------|--------|------|
| GTM 策略 | Intelligence/ | 随市场变化，周期 < 1 年 |
| 年度市场分析 | Intelligence/ | 时效性强，需年度更新 |
| 竞品分析 | Intelligence/ | 竞争态势动态变化 |
| 定价策略 | Intelligence/ | 随市场调整 |
