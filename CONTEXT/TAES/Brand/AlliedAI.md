> **Code in Natural Language. Think in AI Native.**

---

## 1. 学院概况 (About Us)

**Allied AI** 是 **東京アライドAI株式会社 (Tokyo Allied AI K.K.)** 旗下的核心教育事业部，致力于定义 AI 时代的**新语言教育**。

我们认为，未来的通用语言不再仅仅是英语或日语，而是 **Markdown + Prompt**。我们推广以 **VS Code + GitHub** 为核心的 AI 原生工作流，帮助学员掌握与 AI 共生的核心技能。

---

## 2. 核心理念 (Philosophy)

### AI Native = MD + VSC + Copilot

传统的编程教育从语法开始，Allied AI 的教育从**意图 (Intent)** 开始。

- **Markdown (MD)**: AI 的原生语言。结构化你的思想，AI 就能理解你的世界。
- **VS Code (VSC)**: 你的数字工作台。不仅仅写代码，更是写文档、写小说、管理任务的指挥舱。
- **GitHub Copilot**: 你的 AI 结对伙伴。它不是自动完成工具，而是你的思维扩充器。

## 4. 课程与考核 (Curriculum & Certification)

我们不通过死记硬背来考核，而是通过**实战 (Mission)** 与**行业认证**来验证能力。

### 核心目标：GH900 (GitHub Foundations)

所有课程最终指向通过 **GitHub Foundations (GH900)** 认证。这不仅是一张证书，更是你掌握现代协作开发范式的证明。

- **Level 1: Hello Copilot** - 环境搭建，Markdown 基础，Copilot 对话技巧。
- **Level 2: Mission Driven** - 使用 Mission Repo 管理个人项目（写作/编码）。
- **Level 3: Collaboration** - Pull Request, Code Review, Issue 管理。
- **Graduation**: 通过 GH900 考试，获得微软/GitHub 官方认证。

## 5. 顶级教育专家视角的优化方案 (Expert Review & Optimization)

> **评审视角**：基于 Udacity (Nanodegree), Coursera, DeepLearning.AI 的课程设计标准。

### 5.1 理解与洞察 (Insight)

Allied AI 的方案切中了当前 AI 教育的痛点：**工具过剩，方法论匮乏**。
- **优势**：直接提供 M365 + Copilot 商业版 License，解决了学员的“工具门槛”。
- **独特性**：将 Markdown 提升到“语言教育”的高度，非常有前瞻性。
- **商业闭环**：以 GH900 为出口，标准化了交付结果。

### 5.2 优化建议 (Optimization Strategy)

若要达到世界级水准，建议在以下维度进行升级：

#### A. 从“考证”升级为“作品集” (Portfolio Driven)
- **现状**：以 GH900 为终点。
- **优化**：引入 **Mission Repo (MR)** 机制。学员毕业不仅要有证书，还要有一个高星的 GitHub Repo（如一本用 AI 写的小说、一个 AI 辅助开发的网页）。
- **Slogan 升级**：*Don't just get certified. Build your legacy.*

#### B. 引入“AI 结对导师” (AI Peer Mentor)
- **现状**：提供 Copilot 工具。
- **优化**：定制 workspace 级的 `@mentor` Agent（基于 Copilot Instructions）。
- **场景**：当学员在 VS Code 中卡住时，Agent 主动提示：“你现在的 Markdown 结构 AI 可能看不懂，试着加个二级标题？”

#### C. 工业级协作模拟 (Real-world Simulation)
- **现状**：提供 GitHub Org。
- **优化**：课程作业必须通过 **Pull Request** 提交。引入自动化的 **CI/CD 评分系统**（GitHub Actions），对 Markdown 格式、链接有效性进行自动检查，模拟真实大厂的 Code Review 流程。

#### D. 权益货币化 (Tokenomics)
- **现状**：提供 License。
- **优化**：将 Copilot 的使用量（Token）与作业完成度挂钩。完成 Mission 获得更多算力配额，培养学员的“算力成本意识”。
