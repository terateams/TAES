## 一、市场特异性对比

| 维度 | 🇨🇳 中国 | 🇯🇵 日本 |
|------|----------|----------|
| **市场规模（AI 企业服务）** | $8B（2026E） | $3B（2026E） |
| **增速** | 35% | 20% |
| **合规环境** | 《生成式AI管理办法》严格执行 | 《AI事業者ガイドライン》强调透明性 |
| **决策周期** | 快（1-3 月） | 慢（3-6 月） |
| **付费习惯** | 买人天 > 买订阅 | 年度合同优先 |
| **客户心智** | ROI 敏感、降本增效 | 流程合规、风险规避 |
| **渠道** | 直销为主 | SI 渠道为主 |

## 二、中国市场策略

### 2.1 机会与风险

| 维度 | 机会 | 风险 |
|------|------|------|
| **合规** | Workplane 合规链路是刚需 | 政策变化可能影响架构 |
| **竞争** | 差异化在于"方法论 + 执行体系" | 大厂可能推出类似服务 |
| **定价** | Layer 4 指标直击 ROI 痛点 | 价格战压力大 |
| **付费** | 里程碑制符合习惯 | 需要强销售驱动 |

### 2.2 目标客户画像（中国）

| Tier | 客户类型 | 典型规模 | 核心痛点 | 产品层 | 预期 ACV |
|------|----------|----------|----------|--------|----------|
| **S** | 出海创业公司 | 5-15 人 | 快速搭建 AI 工作流；合规访问海外服务 | Starter → Cloud | ¥3-5万/年 |
| **M** | 企业创新部门 | 15-50 人 | AI 落地有 ROI 压力；需要向上汇报 | Cloud | ¥8-15万/年 |
| **EDU** | 教育机构/培训 | 不限 | 课程体系化；学员实操环境 | Starter + 定制 | ¥5-20万/年 |

### 2.3 教育领域深耕策略

#### 为什么选择教育领域？

| 维度 | 优势 | 说明 |
|------|------|------|
| **决策快** | ✅ | 培训机构/高校部门决策链短 |
| **复制性强** | ✅ | 课程模板可批量复用 |
| **口碑传播** | ✅ | 学员 = 未来客户/推荐者 |
| **内容资产** | ✅ | 课程本身可作为 Mission 案例 |
| **竞争弱** | ✅ | AI 方法论培训市场碎片化 |

## 三、日本市场策略

| 维度 | 机会 | 风险 |
|------|------|------|

| **EDU** | 専門学校/企業研修 | 不限 | AI 人材育成 | 教育合作 |

## 四、产品架构（修订版）

> 采用 Layer 1 + Layer 2 架构，放弃 Layer 3（当前实施能力不足）

```
┌─────────────────────────────────────────────────────────────┐
│              TAES 产品矩阵 v2.0（双层架构）                   │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  Layer 2: TAES Cloud (SaaS) ← 核心增长引擎                  │
│  ├── Mission Dashboard（任务看板 + AI 贡献度追踪）           │
│  ├── Eval Platform（评测即服务）                            │
│  ├── Evidence Vault（证据链存储 + 合规报告）                 │
│  ├── 算力配额（并入订阅，不单独定价）                        │
│  └── 定价：¥5,000-10,000/月/团队（10 seats）               │
│                                                             │
│  Layer 1: TAES Starter (入门) ← 获客漏斗                    │
│  ├── Playbook 开源/低价                                     │
│  ├── MR 模板库                                              │
│  ├── 基础课程                                               │
│  ├── 社区支持                                               │
│  └── 定价：免费 / ¥500/月                                   │
│                                                             │
│  ❌ Layer 3: TAES Pro (暂不开放)                            │
│  └── 原因：当前实施能力不足以支撑企业级定制                   │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 4.1 Layer 1: TAES Starter

| 组件 | 内容 | 定价 |
|------|------|------|
| **Playbook** | 完整方法论文档 | 免费（开源） |
| **模板库** | 10+ Mission Repo 模板 | 免费 |
| **基础课程** | 8 课时在线课程 | ¥500/月 或 ¥2,999 买断 |
| **社区** | Discord/微信群 | 免费 |
| **Workplane 试用** | 7 天试用 | 免费 |

**目标**：降低 CAC，建立漏斗顶部流量

### 4.2 Layer 2: TAES Cloud

| 组件 | 内容 | 说明 |
|------|------|------|
| **Mission Dashboard** | 任务看板 + 进度追踪 + AI 贡献度可视化 | 核心产品 |
| **Eval Platform** | 评测即服务：上传 Golden Set → 自动评测 → 报告 | 差异化功能 |
| **Evidence Vault** | 证据链存储 + 合规报告生成 + 审计导出 | 合规刚需 |
| **Workplane** | 稳定 AI 访问（算力配额并入） | 基础设施 |
| **模板市场** | 付费模板 + 社区模板 | 生态建设 |

**定价**：

| 版本 | 席位 | 月费 | 年费（17% 折扣） |
|------|------|------|------------------|
| **Team** | 10 | ¥5,000 | ¥50,000 |
| **Business** | 30 | ¥12,000 | ¥120,000 |
| **Enterprise** | 100 | ¥30,000 | ¥300,000 |

## 五、客户画像（最终版）

> 聚焦 S + M + EDU，放弃 L/XL（当前能力不足）

### 5.1 Tier S：出海创业 / 小型团队

| 维度 | 描述 |
|------|------|
| **规模** | 5-15 人 |
| **典型客户** | SaaS 创业公司、出海电商、独立开发者团队 |
| **核心痛点** | 快速搭建 AI 工作流；合规访问海外 AI 服务；成本敏感 |
| **决策者** | CEO / CTO |
| **决策周期** | 1-2 周 |
| **产品** | Starter → Cloud Team |
| **ACV** | ¥3-5 万 |
| **获客渠道** | 内容营销、社区、课程转化 |

### 5.2 Tier M：企业创新部门

| 维度 | 描述 |
|------|------|
| **规模** | 15-50 人 |
| **典型客户** | 传统企业数字化部门、互联网公司创新团队 |
| **核心痛点** | AI 落地有 ROI 压力；需要向上汇报；合规审计 |
| **决策者** | 部门总监 / VP |
| **决策周期** | 1-3 月 |
| **产品** | Cloud Business |
| **ACV** | ¥10-20 万 |

| **核心痛点** | 课程体系化；学员实操环境；认证体系 |

## 八、风险与缓解

| 风险 | 概率 | 影响 | 缓解措施 |
|------|------|------|----------|
| **教育市场竞争加剧** | 中 | 中 | 聚焦"方法论 + 实操"差异化；建立认证壁垒 |
| **客户转化率低** | 中 | 高 | 优化课程到产品的转化路径；增加试用期 |
| **日本市场进入慢** | 高 | 低 | 优先中国验证；日本作为 Phase 2 |
| **产品开发延迟** | 中 | 高 | MVP 优先；快速迭代 |

---

> **总结**：优先中国市场，以教育为切入点，通过课程获客 → Cloud 转化 → 标杆案例 → 规模复制。日本市场作为 Phase 2，待中国验证后再启动。
