## 市场分析文档规范

### 必须包含的章节

```markdown
# Market Analysis {Year}

## 1. 宏观环境 (Macro Environment)
   - 经济趋势
   - 政策环境
   - 技术趋势

## 2. 市场规模 (Market Size)
   - TAM / SAM / SOM
   - 增长率预测

## 3. 细分市场 (Market Segments)
   - 按行业
   - 按规模
   - 按地理

## 4. 竞争格局 (Competitive Landscape)
   - 主要玩家
   - 市场份额
   - 差异化分析

## 5. 机会与威胁 (Opportunities & Threats)
   
## 6. 战略建议 (Strategic Recommendations)

## Changelog
```

### 命名规范

- 年度报告：`Market-{Year}.md`
- 季度更新：`Market-{Year}-Q{N}.md`
- 专题分析：`Market-{Year}-{Topic}.md`
