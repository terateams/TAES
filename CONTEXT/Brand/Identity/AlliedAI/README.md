## 品牌概述

**Allied AI** (アライドAI) 是 **東京アライドAI株式会社** 旗下的核心教育事业部。

### 品牌定位

> **Code in Natural Language. Think in AI Native.**  
> **自然言葉でコードを紡ぐ。AIネイティブで思考する。**

### 核心理念

- **Markdown (MD)**: AI 的原生语言
- **VS Code (VSC)**: 数字工作台
- **GitHub Copilot**: AI 结对伙伴

## 视觉识别

### 色彩

| 用途 | 颜色 | Hex | 说明 |
|------|------|-----|------|
| 主色 | 蓝染 (Aizome) | `#003366` | 日本传统深蓝 |
| 辅色 | 樱色 (Sakura) | `#FFB7C5` | AI 灵动感 |
| 背景 | 电光白 | `#FAFAFA` | 干净现代 |

### 设计风格

- **东京美学**: 干净线条、留白 (间)、传统与未来融合
- **几何 + 书法**: 代码的几何感 + 自然语言的笔触
- **Zen 风格**: 精确、知性、呼吸感
