## 治理原则

### 1. 最小权限
只授予必要的 AITa 访问权限

### 2. 数据分类
不同敏感级别的数据，不同的 AI 使用策略

### 3. 可追溯
关键操作有日志，可审计

### 4. 持续合规
定期 review，适应变化的合规要求
