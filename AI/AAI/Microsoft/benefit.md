# 🎁 M365 Benefit

> 企业订阅附赠权益

---

## 概述

| 维度 | 说明 |
|------|------|
| **来源** | M365 E3/E5 企业订阅 |
| **权益** | Copilot 席位额度 |
| **用途** | M365 Copilot 分配 |
| **产出** | M365 Copilot AITa |

---

## Benefit 是什么？

```
M365 E3/E5 企业订阅
    ↓
Benefit 资源池 (附赠权益)
    ↓
Copilot 席位额度
    ↓
分配给 E队成员
    ↓
M365 Copilot AITa ✅
```

---

## Benefit 额度结构

| 订阅类型 | Copilot Benefit | 备注 |
|----------|-----------------|------|
| M365 E5 | 20% 席位 | 如 100 E5 → 20 Copilot |
| M365 E3 | 10% 席位 | 如 100 E3 → 10 Copilot |
| 组合 | 按比例计算 | |

> M365 Copilot: $30/席位/月

---

## 分配流程

### 1. 确认 Benefit 池
- [ ] 登录 M365 Admin Center
- [ ] 查看 Benefit 可用额度
- [ ] 确认 Copilot 席位数

### 2. 席位规划
- [ ] 收集 E队需求
- [ ] 优先级排序
- [ ] 分配方案

### 3. 分配执行
- [ ] Admin Center → Licenses
- [ ] 分配 Copilot 许可
- [ ] 通知用户激活

## 席位管理策略

| 优先级 | 角色 | 理由 |
|--------|------|------|
| P0 | 核心文档产出者 | 高频使用 |
| P1 | 项目经理 | PPT/报告需求 |
| P2 | 数据分析师 | Excel Copilot |
| P3 | 一般员工 | 按需分配 |

---

## 监控指标

| 指标 | 说明 |
|------|------|
| 分配率 | 已分配 / 总额度 |
| 激活率 | 已激活 / 已分配 |
| 使用率 | 月活跃 / 已激活 |
| 回收率 | 低使用席位回收 |

---

## 回收机制

```
月使用率 < 10%
    ↓ 连续 2 月
发送提醒
    ↓ 1 月未改善
回收席位，重新分配
```
