# 💳 Azure IUR

> Imagine Use Right - 内部使用权

---

## 概述

| 维度 | 说明 |
|------|------|
| **全称** | Internal Use Rights (IUR) |
| **来源** | Microsoft Partner 权益 |
| **用途** | Azure 订阅额度，可抵扣 Copilot |
| **产出** | GitHub Copilot AITa |

---

## IUR 是什么？

```
Microsoft Partner Program
    ↓
Partner Benefits (合作伙伴权益)
    ↓
Internal Use Rights (IUR)
    ↓
Azure Subscription 额度
    ↓
抵扣 GitHub Copilot 费用
    ↓
GitHub Copilot AITa ✅
```

---

## IUR 额度结构

| 等级 | 年度 Azure 额度 | 可支持 Copilot 席位 |
|------|-----------------|---------------------|
| Solutions Partner | $18,000+ | ~75 席位/年 |
| Gold Partner | $12,000+ | ~50 席位/年 |
| Silver Partner | $6,000+ | ~25 席位/年 |

> GitHub Copilot Business: $19/席位/月 = $228/席位/年

---

## 申请流程

### 1. 确认资格
- [ ] 登录 Partner Center
- [ ] 确认合作伙伴等级
- [ ] 查看可用 IUR 额度

### 2. 激活 Azure 订阅
- [ ] Partner Center → Benefits → Azure
- [ ] 激活 IUR Azure 订阅
- [ ] 记录订阅 ID

### 3. 关联 GitHub
- [ ] Azure Portal → GitHub Copilot
- [ ] 选择 IUR 订阅付费
- [ ] 分配席位

## 监控指标

| 指标 | 阈值 | 动作 |
|------|------|------|
| 月度消耗率 | >10% 额度 | 正常 |
| 季度累计 | >25% 额度 | 正常 |
| 剩余额度 | <20% | ⚠️ 预警 |
| 剩余额度 | <10% | 🚨 紧急 |

---

## 常见问题

**Q: IUR 额度可以累积吗？**
A: 不可以，年度清零。

**Q: IUR 只能用于 Copilot 吗？**
A: 可用于所有 Azure 服务，我们优先用于 Copilot。

**Q: 额度不够怎么办？**
A: 1) 升级合作伙伴等级 2) 购买额外订阅
