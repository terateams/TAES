## 升级为独立 Agent

如需将内置角色升级为独立 Agent（获得更精确的控制）：

1. 在 `.github/agents/` 创建 `{agent-name}.md`
2. 定义角色、能力、使用示例
3. （可选）创建 `.github/skills/{AgentName}/SKILL.md`
4. 更新本注册表

**升级候选**（使用频率高、需要专业化）：
- [ ] `@mentor` — 可加入公司文化、晋升路径等上下文
- [ ] `@tech` — 可加入技术栈偏好、代码规范
- [ ] `@product` — 可加入产品方法论、竞品分析
