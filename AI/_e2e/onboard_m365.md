# 📝 M365 Copilot 上手指南

> 从 0 到能用，预计 5 分钟

---

## 前置条件

| 条件 | 检查方式 | 未就绪怎么办 |
|------|----------|--------------|
| M365 企业账号 | 能登录 office.com | 联系 IT |
| M365 E3/E5 许可 | 能用 Word/Excel/PPT | 联系 IT |

---

## 步骤

### Step 1: 确认席位分配 (2分钟)

1. 联系 **AITa 管理员**（或在内部系统申请）
2. 确认你已被分配 M365 Copilot 席位
3. 收到确认邮件/消息

> 💡 席位来源：M365 Benefit 红利，你无需付费

### Step 2: 等待许可生效 (0-24小时)

- 管理员分配后，许可同步需要时间
- 通常 15 分钟内生效
- 最长 24 小时

### Step 3: 验证激活 (1分钟)

**Web 版：**
1. 打开 [office.com](https://office.com)
2. 登录企业账号
3. 打开 Word Online
4. 看到 Copilot 图标 ✅

**桌面版：**
1. 打开 Word / Excel / PowerPoint
2. 确认已登录企业账号
3. 功能区看到 Copilot 按钮 ✅

### Step 4: 首次使用 (2分钟)

**Word:**
1. 新建文档
2. 点击 Copilot 图标
3. 输入："帮我写一份项目周报模板"
4. 查看生成结果 🎉

**PowerPoint:**
1. 新建演示文稿
2. 点击 Copilot
3. 输入："创建一个产品发布会 PPT，5 页"

**Excel:**
1. 打开数据表
2. 选择数据区域
3. 点击 Copilot
4. 输入："分析这些数据的趋势"

## 支持的应用

| 应用 | Copilot 能力 |
|------|-------------|
| **Word** | 起草、改写、总结 |
| **Excel** | 公式、分析、图表 |
| **PowerPoint** | 创建、设计、总结 |
| **Outlook** | 邮件起草、总结长邮件 |
| **Teams** | 会议总结、聊天摘要 |
| **OneNote** | 笔记整理、生成内容 |

## 遇到问题？

| 问题 | 解决 |
|------|------|
| 看不到 Copilot 按钮 | 等待许可同步 / 重新登录 |
| 提示无权限 | 联系管理员确认席位 |
| Web 版有，桌面版没有 | 更新 Office 到最新版 |
| Copilot 响应慢 | 正常，复杂任务需要时间 |

→ 更多问题见 [troubleshoot.md](troubleshoot.md)
