# 📋 服务等级承诺 (SLA)

> AITa 服务保障

---

## 服务范围

| 层次 | 我们承诺 | 我们不承诺 |
|------|----------|------------|
| **AITa 可用性** | 席位分配、账号就绪 | AI 本身的正常运行 |
| **支持响应** | 工作日 4 小时内响应 | 7×24 即时响应 |
| **问题解决** | 尽力协助 | 保证解决所有问题 |

---

## AITa 服务承诺

### GitHub Copilot

| 指标 | 承诺 |
|------|------|
| **席位分配** | 申请后 1 个工作日内 |
| **激活支持** | 工作日 4 小时内响应 |
| **服务可用性** | 依赖 GitHub 服务状态 |

> ⚠️ GitHub 服务中断时，我们无法解决，请关注 [status.github.com](https://status.github.com)

### M365 Copilot

| 指标 | 承诺 |
|------|------|
| **席位分配** | 申请后 1 个工作日内 |
| **许可生效** | 分配后 24 小时内 |
| **激活支持** | 工作日 4 小时内响应 |
| **服务可用性** | 依赖 Microsoft 365 服务状态 |

> ⚠️ M365 服务中断时，我们无法解决，请关注 [status.office.com](https://status.office.com)

### Google Gemini

| 指标 | 承诺 |
|------|------|
| **账号分配** | 申请后 1 个工作日内 |
| **激活支持** | 工作日 4 小时内响应 |
| **服务可用性** | 依赖 Google 服务状态 + Workplane |

> ⚠️ 网络可达性依赖 Workplane，请确认 CONNECTION 就绪

## 支持响应时间

| 优先级 | 描述 | 响应时间 |
|--------|------|----------|
| **P0 紧急** | 所有人无法使用 | 1 小时内 |
| **P1 高** | 部分人无法使用 | 4 小时内 |
| **P2 中** | 功能异常但可用 | 1 个工作日 |
| **P3 低** | 咨询、建议 | 2 个工作日 |

---

## 责任边界

### 我们负责 ✅

| 事项 | 说明 |
|------|------|
| 席位采购 | 通过红利路径获取 |
| 席位分配 | 按需分配给 E队 |
| 账号管理 | 开通、回收 |
| 使用指导 | 提供文档和支持 |
| 问题协助 | 排查和协调 |

### 我们不负责 ❌

| 事项 | 原因 |
|------|------|
| AI 服务本身中断 | 依赖第三方（GitHub/Microsoft/Google） |
| AI 输出质量 | AI 能力由大厂决定 |
| 网络可达性细节 | 依赖 CONNECTION/Workplane |
| 个人设备问题 | 需要 IT 支持 |

## 服务中断处理

### 当大厂服务中断时

```
发现中断
    ↓
确认是大厂问题（查状态页）
    ↓
内部通知（说明情况）
    ↓
持续监控
    ↓
恢复后通知
```

### 状态页

| AITa | 状态页 |
|------|--------|
| GitHub Copilot | status.github.com |
| M365 Copilot | status.office.com |
| Google Gemini | status.cloud.google.com |
