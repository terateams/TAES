# 🔧 常见问题排查

> AITa 使用问题自助解决

## 通用问题

### Q: 不知道该用哪个 AITa？

```
写代码？ → GitHub Copilot
    ↓ 不是
写文档/PPT？ → M365 Copilot
    ↓ 不是
长文本/深度分析？ → Google Gemini
```

### Q: 如何申请 AITa 席位？

1. 联系 AITa 管理员
2. 或在内部系统提交申请
3. 说明需要哪个 AITa + 用途
4. 等待审批和分配

### Q: 为什么我要付费？

你不需要付费！所有 AITa 成本由公司承担：
- GitHub Copilot → Azure IUR 红利
- M365 Copilot → M365 Benefit
- Google Gemini → TAES 代付

如果提示付费，说明席位未正确分配，请联系管理员。

## GitHub Copilot 问题

### Q: 安装插件后没有代码建议

**检查清单：**
- [ ] 确认已登录 GitHub 账号
- [ ] 确认账号已被分配 Copilot 席位
- [ ] 检查 VS Code 右下角 Copilot 状态图标
- [ ] 尝试重启 VS Code

**状态图标含义：**
| 图标 | 状态 |
|------|------|
| ✅ 绿色 | 正常工作 |
| ⚠️ 黄色 | 有警告 |
| ❌ 红色/灰色 | 未激活/错误 |

### Q: 提示 "You don't have access to GitHub Copilot"

1. 确认你的 GitHub 账号
2. 联系管理员确认席位分配
3. 检查是否在正确的 GitHub Organization

### Q: Copilot Chat 无响应

1. 检查网络连接
2. 尝试重新登录 GitHub
3. 检查是否有防火墙限制
4. 重启 VS Code

### Q: 建议质量不好

1. 提供更多上下文（注释、函数名）
2. 打开相关文件让 Copilot 学习
3. 使用 `@workspace` 让 Copilot 理解项目
4. 尝试 Copilot Chat 详细描述需求

## M365 Copilot 问题

### Q: Office 里看不到 Copilot 按钮

**检查清单：**
- [ ] 确认许可已分配（可能需要等待同步）
- [ ] 确认 Office 版本是最新的
- [ ] 尝试重新登录 M365 账号
- [ ] 尝试 Web 版 office.com

**许可同步时间：**
- 通常 15 分钟内
- 最长 24 小时
- 如超过 24 小时，联系管理员

### Q: Copilot 按钮灰色不可用

1. 可能是网络问题，稍后重试
2. 检查是否登录了正确的企业账号
3. 尝试重启 Office 应用

### Q: Copilot 生成内容不准确

1. 提供更详细的提示
2. 指定格式、风格、长度
3. 使用"改进"功能迭代
4. 对于数据分析，确保数据格式正确

### Q: Teams 会议没有 Copilot 总结

1. 确认会议已开启录制
2. 会议结束后需要等待处理
3. 检查是否有 Teams Copilot 权限

## Google Gemini 问题

### Q: 无法访问 gemini.google.com

**检查清单：**
- [ ] 确认 Workplane 网络正常
- [ ] 尝试其他浏览器
- [ ] 清除浏览器缓存
- [ ] 联系网络管理员

### Q: 登录后不是 Gemini Advanced

1. 确认使用的是代付账号（不是个人账号）
2. 检查账号邮箱是否正确
3. 联系管理员确认账号状态

### Q: 文件上传失败

**限制：**
- 单文件最大：部分格式有限制
- 支持格式：PDF, TXT, DOC, 图片等
- 如太大，尝试分割文件

### Q: 回复被截断

1. 长回复可能分段
2. 输入"继续"让 Gemini 继续
3. 或分多个问题提问
