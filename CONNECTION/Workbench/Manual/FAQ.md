## 基础操作

### Q: 如何编辑文件？
A: 两种方式：
1. **网页编辑**：点击文件 → 点击铅笔图标 ✏️ → 编辑 → Commit
2. **VS Code**：在 VS Code 中打开仓库直接编辑

### Q: 我不小心改错了怎么办？
A: 不用担心！GitHub 保存了所有历史版本，可以随时恢复。联系管理员帮你处理。

### Q: 什么是 Commit？
A: Commit 就是"提交保存"，每次 Commit 都会记录你的修改，可以随时回溯。

## 任务相关

### Q: 如何知道我的任务是什么？
A: 
1. 查看 Issues 标签页
2. 筛选 `assignee:你的用户名`
3. 或者查看 Projects 看板

### Q: 如何标记任务完成？
A: 在 Issue 中留言说明已完成，或者直接关闭 Issue。

### Q: 任务有问题/需要帮助怎么办？
A: 在 Issue 中 @相关人员，或在 Discussions 中提问。

## Copilot 相关

### Q: Copilot 没有响应？
A: 检查：
1. 是否已安装 GitHub Copilot 扩展
2. 是否已登录 GitHub 账号
3. 账号是否有 Copilot 权限

### Q: Copilot 给的答案不对？
A: AI 可能出错，需要你验证。可以：
1. 换一种方式提问
2. 提供更多上下文
3. 向团队成员确认
