# GitHub Copilot 使用指南

> 预计学习时间：10 分钟

## 什么是 GitHub Copilot？

GitHub Copilot 是 AI 编程助手，可以：
- 💬 回答问题
- 📝 生成文档
- 🔍 分析代码
- 💡 提供建议

## 如何使用

### 1. 在 VS Code 中

1. 安装 GitHub Copilot 扩展
2. 登录 GitHub 账号
3. 点击侧边栏的 Copilot 图标
4. 开始对话

### 2. 快捷键

| 功能 | 快捷键 |
|------|--------|
| 打开 Copilot Chat | `Ctrl + Shift + I` |
| 内联建议 | `Tab` 接受 |

## 使用示例

### 示例 1：分解任务

```
请以项目经理的身份，帮我分解这个任务：

任务：开发一个内部审批系统
截止时间：2周后
团队：3人

请给出子任务列表和时间安排。
```

### 示例 2：学习问题

```
请以同学的身份，和我讨论一下：

我最近在学习 GitHub 的使用，感觉 Branch 和 PR 的概念有点绕，能帮我理一理吗？
```

### 示例 3：技术问题

```
请以技术专家的身份，帮我分析：

这段代码运行很慢，可能是什么原因？应该怎么优化？

```代码...```
```

## 提示模板

本仓库还提供了一些提示模板：

| 模板 | 用途 |
|------|------|
| `task-breakdown` | 任务分解 |
| `code-review` | 代码审查 |
| `doc-generate` | 文档生成 |
| `meeting-notes` | 会议纪要 |
| `progress-report` | 进度报告 |

## 小贴士

1. **问题要具体**：给出足够的背景信息
2. **指定角色**：让 AI 以特定身份回答
3. **分步骤问**：复杂问题拆成小问题
4. **验证结果**：AI 可能出错，需要核实
