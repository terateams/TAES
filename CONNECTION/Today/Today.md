# 第一部：GMO 能力完善

## 2. GMO + Wise 集成操作

### 2.1 合作本质

GMO 与 Wise 的合作为深度 API 集成（Embedded Finance），可直接在 GMO 网银后台使用 Wise 国际汇款引擎。

### 2.2 核心优势

| 维度 | 说明 |
|------|------|
| **汇率** | 中间市场汇率，无隐藏汇差 |
| **手续费** | 较传统 SWIFT 节省 70%~90% |
| **速度** | 小时级到账，部分秒到 |
| **操作** | 无需离开 GMO 网银页面 |

### 2.3 操作路径

**首次联动**：
1. 登录 GMO 法人网银
2. 进入 `外貨・海外送金` → `海外送金 (Wise)`
3. 完成联动申请及授权绑定

**日常汇款**：
1. 输入汇款金额和目标币种
2. 确认到账金额、手续费、预计到账时间
3. 选取收款人，确认扣款

### 2.4 Wise Business 独立账户（进口收款）

建议独立注册 Wise Business 账户，获取美元/欧元/英镑本地银行收款账号：
- 海外客户汇入 Wise 本地账户（零手续费）
- 批量转回 GMO 账户，或留存支付海外费用

## 5. 双银行架构设计

### 5.1 功能分工

| 银行 | 定位 | 主要职能 |
|------|------|----------|
| **GMO あおぞらネット銀行** | 主营业务账户 | 日常 IT 业务结算、云服务扣费、国际支付（via Wise） |
| **乐天银行** | 储备与支付账户 | 海外大额收款、税金/社保缴纳、员工工资发放 |

### 5.2 资金路由架构

```
┌─────────────────────────────────────────────────────────────────┐
│  Allied Tokyo 资金路由架构                                       │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  国内业务线：                                                   │
│  GMO (主营) ←───── 内部划转 (振替) ─────→ 乐天 (储备/工资)      │
│                                                                 │
│  国际支付线（出口）：                                           │
│  GMO ──→ Wise API ──→ 海外付款 (USD/EUR/CNY)                   │
│                                                                 │
│  国际收款线（进口）：                                           │
│  海外客户 ──→ Wise 本地账号 ──→ GMO / 乐天                      │
│              或 乐天 SWIFT 直汇                                 │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 5.3 冗余备份价值

- 若 GMO 系统维护或因审查暂时冻结 → 乐天立即接管支付流程
- 若乐天维护 → GMO 接管
- 确保 Allied Tokyo 运营不中断

## 8. 财务合规要点（双银行场景）

### 8.1 税务申报

- 所有银行账户的期末余额、利息收入须完整体现在《科目内訳書》中
- 隐藏任何账户将被视为偷税风险

### 8.2 会计同步

- 将两个账户同时挂载到 Money Forward Cloud 或 freee
- 两账户间的内部资金划转须标记为"振替"（Transfer），避免账目虚高

### 8.3 反洗钱规范

- 避免账户余额频繁清零
- 在乐天账户中长期保留沉淀资金（建议 100-200 万日元），有助于提升公司信用等级
- 自转账备注写明 `資金移動` 或 `給与資金`

### 8.4 访问安全

- GMO 使用手机 App 确认
- 乐天可能提供物理令牌或数字证书
- 建议将认证权限分设在不同物理设备上
